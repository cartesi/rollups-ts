# cartesi_listOutputs

## Usage

```ts twoslash
// [!include ~/snippets/rpc/createClient.ts]

const { data, pagination } = await client.request("cartesi_listOutputs", {
    application: "0x0974CC873dF893B302f6be7ecf4F9D4b1A15C366",
    limit: 50,
    offset: 0,
});
```

## Filters

The request can filter results by:

- `epoch_index: HexNumber`
- `input_index: HexNumber`
- `output_type: HexNumber`
- `voucher_address: Address`

```ts twoslash
import { type Address, type HexNumber } from "@cartesi/rpc"
```

## Return Type

The `data` is an array of `Output`:

```ts twoslash
import {
    type Output,
    type ListOutputsReturnType,
} from "@cartesi/rpc";
```

For more information about pagination refer to the [pagination section](/rpc#pagination).
