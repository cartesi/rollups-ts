# cartesi_listEpochs

## Usage

```ts twoslash
// [!include ~/snippets/rpc/createClient.ts]

const { data, pagination } = await client.request("cartesi_listEpochs", {
    application: "0x0974CC873dF893B302f6be7ecf4F9D4b1A15C366",
    limit: 50,
    offset: 0,
});
```

## Filters

The request can filter results by:

- `status: EpochStatus`

```ts twoslash
import { type EpochStatus } from "@cartesi/rpc"
```

## Return Type

The `data` is an array of `Epoch`:

```ts twoslash
import {
    type Epoch,
    type ListEpochsReturnType,
} from "@cartesi/rpc";
```

For more information about pagination refer to the [pagination section](/rpc#pagination).
